# cpp-stroustrup-basics

Collection of small, self-contained C++ console programs, each .cpp file illustrating a specific C++ concept (I/O, random numbers, pointers/refs, conversions, constexpr, etc.), based on Bjarne Stroustrup's _Programming: Principles and Practice Using C++_ (3rd ed.), and occasionally other sources as noted.

The core insights for each snippet are provided as comments where available.

These _are not_ snippets intended to show either the book's drills or the chapter exercises --other sources for this exist abundantly elsewhere in the web. These _are_ snippets intended to distill and illustrate core concepts of the book as I explore them. 

Any errors are of course mine.
